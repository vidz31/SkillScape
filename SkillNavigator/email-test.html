📧 Email Configuration Test

Test your SMTP settings to ensure emails are working correctly

Before Testing:

  1. Update appsettings.json with your email credentials
  2. For Gmail: Use an App Password (not your regular password)
  3. Restart the backend server after updating settings
  4. Make sure the backend is running on http://localhost:5000

Current Configuration

📁 backend/SkillScape/SkillScape.API/appsettings.json

⚠️ Common Issues:

  • Still using placeholder values? Replace "your-email@gmail.com" and "your-app-password"
  • Using regular Gmail password? Won't work! You need an App Password
  • No 2FA enabled? App Passwords require 2-Factor Authentication
  • Forgot to restart backend? Changes require a restart to take effect
  • Port 587 blocked? Check firewall/antivirus settings
